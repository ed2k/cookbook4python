This is source code of Floater, a network bridge game.  For
information about Floater, see the "documentation" subdirectory, or
visit:

	http://www.floater.org


To compile Floater from this directory tree, read the appropriate file
in the "documentation/build" directory.


If you have any questions or comments, there are several feedback
mechanisms available on the SourceForge site:

	http://www.sourceforge.net/projects/floaterbridge

Feel free to browse the forums, email the maintainers, or post to the
bug tracker.


The subdirectories of this directory are:

  art
      Files containing artwork, including icons and card bitmaps.

  build
      Files used to build the progam, such as Makefiles and autoconf
      scripts.

  common
      Files common to many of the Floater binaries.  This also includes
      the Unix client for both Tk and ncurses.

  debian
      Files for build debian packages.

  documentation
      All kinds of documentation, including FAQ's, tutorials, change
      logs, and legal information.

  loginserver
      The login server.

  pseudomailer
      The "pseudomailer", a daemon that relays mail from Floater clients.

  res2html
      (unsure)

  resultserver
      The result server.

  serverfiles
      Files necessary for running the server.
